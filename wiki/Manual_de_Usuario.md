# Manual de Usuario — pfQuest Sequito Edition 📖

## Uso Básico
1. **Activar Rastreo**: Abre tu diario de misiones (tecla `L`). Verás botones adicionales en la parte superior.
2. **La Flecha**: Haz clic en el botón `[Arrow]` de una misión para que la flecha de navegación se bloquee a ese objetivo.
3. **El Tracker**: El panel lateral muestra tus misiones activas y el progreso. Haz clic en una misión para desplegar sus objetivos.

## Atajos de Teclado (Click-Modifiers)
- **Shift+Click** (Icono de Mapa): Elimina el nodo del mapa (útil para limpiar objetivos completados).
- **Ctrl+Click** (Icono de Mapa): Cambia el color del spawn para mejor visibilidad.
- **Right-Click** (Icono de Tracker): Abre el diario directamente en esa misión.

## Comandos de Chat
- `/db search <término>`: Busca en la base de datos global.
- `/db tracker`: Muestra/Oculta el rastreador de misiones.
- `/db clean`: Limpia todos los nodos temporales del mapa.
